## [0.1.0] - 2025-07-01

### Added
- Initial release of PyMIRT
- Single-dimensional IRT models (2PL, GRM)
- Multi-dimensional IRT models (M2PL, MGRM)
- EM algorithm for parameter estimation
- MCMC methods for parameter estimation
- EAP ability estimation
- Basic documentation and examples

### Core Features
- `irt()` function for single-dimensional IRT models
- `mirt()` function for multi-dimensional IRT models
- Support for binary and polytomous items
- Flexible quadrature settings
- Convergence monitoring
- Verbose output options

### Supported Models
- 2PL (Two-Parameter Logistic Model)
- GRM (Graded Response Model) - Standard and Stepwise
- M2PL (Multi-dimensional Two-Parameter Logistic Model)
- MGRM (Multi-dimensional Graded Response Model)

### Estimation Methods
- Expectation-Maximization (EM) algorithm
- Markov Chain Monte Carlo (MCMC)
- Expected A Posteriori (EAP) ability estimation
